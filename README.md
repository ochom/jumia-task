# jumia-task
A simple rest and UI app

This task has two parts

- backend (golang)
- frontend (react)

## Running the application

### Backend
- `cd` into project root
- run
  ```sh
  # install all go modules
  $ go mod tidy
  $ go run .
  ```

## Frontend
- `cd` into project/frontend
- run

  ```sh
  $ yarn install -y
  $ source env.sh
  $ yarn start
  ```
  
- head over to http://localhost:3000/ on your browser
